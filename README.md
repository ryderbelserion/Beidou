# Beidou
A bot written in `Java`, primarily used for my personal servers that I own, or help maintain.

# How can I self host?
1) Clone the project
2) ./gradlew shadowJar
3) java -jar beidou.jar --token <your_token>
4) Change the whitelisted guilds in cache.json, as this bot is meant to be multi-guild.
